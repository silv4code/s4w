# Security Policy

## Authorized Use Only

S4W is a passive web security assessment toolkit intended for owned assets, controlled labs, internal reviews and formally authorized environments.

Do not use this project against third-party systems without explicit permission.

## Out of Scope by Design

This project does not accept features that automate:

- credential brute force;
- password spraying or credential stuffing;
- exploit delivery;
- authentication bypass;
- destructive testing;
- stealth, persistence or evasion.

Passive checks, documentation, structured reporting and defensive hardening guidance are in scope.

## Reporting Security Issues

If you find a vulnerability in S4W itself, please use GitHub private vulnerability reporting or contact the maintainer privately before public disclosure.
